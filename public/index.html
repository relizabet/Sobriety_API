Sobriety Quotes API

An API for Sobriety Quotes

The sobriety quotes API was inspired by and created for the Ascent application when it became apparent that there was a lack of quote API's that were appropriate for and tailored to people that are sober and in recovery.

Allowed HTTPs requests:

GET: To get quotes

Get one single random quote:
  • https://desolate-caverns-92812.herokuapp.com/api/random
Get all quotes:
  • https://desolate-caverns-92812.herokuapp.com/api/quotes

POST: To submit quotes

Get one single random quote:
  • https://desolate-caverns-92812.herokuapp.com/api/add-quote

Server Responses

  • 200 OK - request successful
  • 201 CREATED - quote successfully submitted
  • 400 BAD REQUEST - request could not be completed
